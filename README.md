
# Speed Camera Visualizations - Transportation Data Project

This project was hosted by the Northeast Big Data Innovation Hub & National Student Data Corps, in collaboration with the U.S. Department of Transportation Federal Highway Administration.

---

## Project Overview

This project analyzes traffic violations across Chicago using open datasets, focusing on speed cameras, red lights, and violation locations. It provides **interactive visualizations** to identify hotspots and understand patterns of traffic enforcement.  

Two complementary maps were created:  

1. **Heatmap** – Aggregates violation points to show **areas with the highest density of traffic violations**.  
2. **Severity Map** – Displays **actual locations of cameras, red lights, and sampled violations**, color-coded for easy interpretation.  

These visualizations help explore urban traffic patterns and enforcement effectiveness in a clear, interactive way so we can apply our findings to other cities and urban areas.

---

## Key Highlights

- Processed and visualized 2M+ entries using **Python**, **Pandas**, **Matplotlib**, and **Seaborn**.  
- Cleaned and preprocessed datasets for meaningful insights.  
- Applied **time-series, bar, line, and geo-spatial analyses** to assess speed camera efficiency.  
- Collaborated with a team of two to deliver actionable findings.
